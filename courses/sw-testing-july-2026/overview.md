# Course Overview

Modern Software Testing with AI-Assisted Automation & CI/CD Integration is a 4-day hands-on training program that equips participants with modern software testing skills by combining automation frameworks, AI-assisted testing techniques, and CI/CD integration.

Who should attend:
- QA Engineers
- Test Engineers
- Software Developers
- Automation Engineers
- DevOps Engineers
- Technical Leads

Prerequisites:
- Basic software testing knowledge
- Familiarity with web applications
- Basic programming knowledge

No prior Playwright or automation experience is required. The course is suitable for beginners with basic software testing and programming familiarity.

Topics covered:
- Playwright automation framework (taught from scratch)
- TypeScript fundamentals (variables, arrays, interfaces, classes, error handling, OOP)
- AI-assisted testing using ChatGPT and GitHub Copilot
- UI automation and API testing
- Page Object Model design pattern
- CI/CD integration with Azure DevOps
- BDD with Cucumber
- End-to-end automation implementation project

Playwright is a modern automation framework for testing web applications across multiple browsers. It provides reliable browser automation, integrated waiting, and API testing support — a strong modern alternative to Selenium.

By the end of the course, participants will be able to design, build, maintain, and deploy modern automated testing solutions using Playwright, AI tools, API testing, and CI/CD practices.

Participants may bring their own laptop or use a Timmins-provided laptop where available.

Q5. What is Playwright?
Answer:
Playwright is Microsoft's modern web automation framework for testing applications across Chromium, Firefox, and WebKit browsers.
It provides reliable browser automation, API testing capabilities, automatic waiting mechanisms, and strong support for CI/CD automation.
Q6. Why is Playwright used instead of Selenium?
Answer:
Playwright provides several modern capabilities including:
Automatic waiting
Better handling of dynamic web applications
Cross-browser testing
Integrated API testing
Faster execution
Improved reliability
The course focuses on Playwright because it aligns with modern QA and DevOps practices.
